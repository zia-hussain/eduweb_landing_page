### Run Locally

To run **EduWeb** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/zia-hussain/eduweb_landing_page.git
```

Windows:

```bash
git clone https://github.com/zia-hussain/eduweb_landing_page.git
```
